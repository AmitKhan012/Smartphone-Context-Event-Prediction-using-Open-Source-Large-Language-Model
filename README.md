# Smartphone-Context-Event-Prediction-using-Open-Source-Large-Language-Model
Modern smartphones function in highly dynamic environments where contextual events such as Wi-Fi connectivity, location, screen state, and battery level significantly influence device performance and reliability. Accurately predicting future context states can ensure comprehensive software testing and in-field optimization. In this work, we investigate the use of large language models (LLMs) for contextual event prediction. Particularly, we fine-tune the Llama-3.2-1B model to predict the next contextual event from preceding event sequences. Experi-
mental evaluation shows that the proposed model achieves over 70% precision, recall, and weighted F1 score and outperforms traditional sequential deep learning models with a higher macro-F1 score (34%). These results demonstrate the prospect of applying LLMs to smartphone context prediction and highlight their potential to enhance mobile system stability and application performance through predictive context modeling.
